Components

Button · Primary

The default action button — used for the loudest affordance on a screen ("Start Session," "Save," "Log rep"). Capsule shape, gradient fill, glow on press.

Button · Secondary

The supporting action — pairs with button-primary for any screen that has more than one affordance. Outlined, tinted, inverts on press. Same capsule shape and 56px tap target as primary so they read as siblings.

Card

Dark raised surface for chart containers, tables, and info panels. Combines color-bg-raised background, color-border-subtle border, and radius-md rounding with comfortable inner padding.

Card content

Chip

The small, dense tag used for skill wheel dimensions, session phases, and filter pills. Capsule shape, tinted soft background. Three variants: default static tag, tap-sized interactive toggle, and selected filter state.

Form Field

A dark-mode form input with a separate uppercase label above, an input row, and helper text below for hints or errors. Five states shown: empty (default), focused (accent border + soft glow), filled (has a value), error (red border + red helper), and a long-form textarea in the focused state.

Score Bars

Horizontal bar chart for per-attribute 1–10 scores. Each row pairs an attribute name + colour swatch with a muted track, a coloured fill segment encoding the score, and a monospace numeric readout. Tick marks sit at 0 / 2.5 / 5 / 7.5 / 10 with an axis row below. Swap the fill background to any color-wheel-* token per attribute category. An optional .prev-marker inside the track shows a previous-cycle score for comparison.

Section Label

Uppercase eyebrow label with a hairline rule extending to the right. Used to group sections in dark-surface documentation layouts. Composed with color-text-muted, tight letter-spacing, and a ::after pseudo-element for the rule.
